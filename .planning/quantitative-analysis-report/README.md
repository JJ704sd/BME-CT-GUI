# 影像量化分析规划目录

本目录记录“影像量化分析与标准化报告”功能的规划、实现进度和计算说明。

## 文件说明

- `task_plan.md`：功能规划、第一阶段范围、实现步骤、测试计划和风险护栏。
- `progress.md`：当前实现进度、已完成内容和后续待办。
- `quantification_explanation.md`：各量化指标的计算公式、当前边界、标准化报告接入方式，以及后续精度/速度优化路径。

## 当前阶段结论

第一阶段采用纯前端 CPU 量化方案：前端拿到分割结果 NIfTI mask 后，基于 label 体素分布和 spacing 计算器官体积、体素数、最大轴向截面积、包围盒最长径和头足向长度估算，并写入评估面板和 HTML/JSON/PDF 报告。

该阶段不修改后端推理、SSE、缓存、validation 或 job summary 逻辑。
