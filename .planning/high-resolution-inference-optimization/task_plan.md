## Phase 3：优化方案评估 [待开始]

### 方案 1：预降采样

**可行性：** 高

**实现方式：**

```python
# 在推理前对高分辨率 CT 进行降采样
import numpy as np
from scipy.ndimage import zoom

# 假设 ct_data shape 为 (768, 768, 103)
zoom_factors = (512/768, 512/768, 1.0)
ct_downsampled = zoom(ct_data, zoom_factors, order=1)
```

**预期效果：** 推理时间减少约 50%（从 90 分钟降到约 40 分钟）

**缺点：** 可能丢失细节信息，需要在结果回填时上采样到原始分辨率

**缺点：** 需要重新训练或获取预训练模型
